변수 - 값을 여러번 재사용을 용이하게 하기 위해서 사용
     - 한 프로그램내에서 특정 값이 일관성있게 유지시킨다.
     - 코드의 가독성이 좋아진다.

함수 - 연속적으로 이어지는 여러행의 코드들을 재사용을
       용이하게 하기 위해서 사용
     - 한 프로그램네에서 연속적으로 이어지는 여러행의
       코드들을 일관성있게 유지시킨다.
     - 코드의 가독성이 좋아진다.

함수의 구조 - 연속적으로 이어지는 코드들을 항상 동일 순서로 진행되고, 코드드을 중에
              어떤 코드도 빠지거나 추가할 필요가 없는 코드들의 묶음으로 함수화한다.

def add(a, b):
    result = a + b
    return result

x = 7
y = 4
z = add(x, y)		1. add(x, y) 함수를 호출하면, 변수(x, y)를 조회하여
			   add(7, 4) 로 수정한 후에 함수를 호출한다.
			   z = add(x, y)	->	z = add(7, 4)
			2. add() 함수가 호출되면, 인자(7, 4)를 매개변수(x, y)에
			   순서대로 대입한다.
			   def add(a, b):	->	def add(a = 7, b = 4):
			       result = a + b		    result = 7 + 4
			       return result		    return 11
			3. return 값 으로 끝나면, 함수가 호출된 자리에 값을 되돌려주면
			   함수는 사라지고, 함수 자리에 값으로 대체된다.
			   z = add(7, 4)	->	z = 11
