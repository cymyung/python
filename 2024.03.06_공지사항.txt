 - 1일 3시간씩, 총 20일 진행 (총 60시간)
 - 수업시간: 19:20(오후) ~ 22:10(오후)
 - 19:30 까지는 정상출석으로 인정되고, 19:31부터 지각처리가 됩니다.
 - 20:45까지 오시면 지각처리가 가능하나 이후에는 결석으로 처리됩니다.
 - 하루 수업시간에 50%이상 수강하셔야 지각/조퇴/외출이 가능합니다.
   19:20까지 오셨으면, 20:50부터 조퇴가 가능합니다.
   19:30에 오시면, 21:00부터 조퇴가 가능합니다.
 - 지각/조퇴/외출이 3회 누적되면 결석 1회로 간주됩니다.
 - 전체 수업시간에 80%이상 하셔야 수료 가능 (일수 계산됨)
   결석 4회까지는 수료가 가능하지만, 결석 5회시 수료가 불가능합니다.

프로그래밍 언어 종류

 1. compile(번역)     : C, Java, ...
    Source Code    -compile(번역)->	Binary Code(저장)  -exe(실행)->	결과(정보)
    사람이 이해				기계가 이해
    하는 코드				하는 코드

 2. interpreter(통역) : Python, Ruby, JavaScript
    Source Code(저장)  -interpreter(통역),exe(실행)->	결과(정보)
    사람이 이해
    하는 코드

버전(Version)
 x.y.z
 x	: Major, 주버전  , 획기적인(큰) 변화
   y	: Minor, 부버전  , 기능 개선(추가)
     z	: Patch, 패치버전, 문제점 해결

IDLE(Intergrated Development Learning Environment)
  - 통합 개발 공부 환경, 편집기
  1. Visual Studio	: 통합, Microsoft 제품, 상용
  2. Visual Studio Code	: 통합, Microsoft 제품, 무료	<- 추천

  3. Anaconda		: 파이썬 전용, 무료/상용, 데이터 분석/머신
			  러닝에 특화된 모듈들을 제공
  4. PyCharm		: 파이썬 전용, 무료/상용, 프로젝트(웹 개발)
			  에 특화된 모듈들을 제공
  5. Notepad++		: 통합, 무료, 메뉴형식으로 간단한 기능을 제공 (설치)
  6. Jupyter Notebook	: 파이썬 전용, 무료, 웹브라우저 환경에서 작업,
			  interactive/editor 환경 모두 지원함.


interactive(대화형) 모드 특징
  - 코드를 1행씩 실행하면서 결과를 확인할 수 있다.
  - Python을 공부할 때, 유용하게 사용할 수 있다.
  - 모드를 종료하면, 모든 작업내용이 소멸된다. (저장X)
  - prompt(프롬프트)
    >>>

command(실행) 창 특징
  - Windows 명령어를 실행할 수 있다.
  - Python (코드)파일을 실행할 수 있다.
  - prompt(프롬프트)
    C:\Users\사용자\...>

IDLE 작업 환경 특징
  - 전체 코드를 다 작성하고, 실행해야 결과를 확인할 수 있다.
  - 작업내용을 저장하면, 다음부터는 여러번 반복적으로 재사용이 가능한다.

프롬프트(Prompt)
  - 사용자에게 현재 어디서 작업중이며, 어떤 일을 할 수 있다는 것을 알려주는 정보(모양)

    C:\Users\Administrator>	- 윈도우즈에서 사용자 홈디렉토리에서
				  작업중이며, Windows 명령어를 입력가능

    >>>				- 현재 파이썬 interactive(대화형) 모드에
				  접속해 있으며, Python Code 작성가능

    ...				- 현재 파이썬 interactive(대화형) 모드에서
				  여러행으로 이루어진 코드를 작성중
